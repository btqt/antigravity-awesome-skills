## Mới đến đây? Bắt đầu tại đây!

**Chào mừng đến với Phiên bản V5.0.0 Workflows.** Đây không chỉ là một danh sách các script; nó là một hệ điều hành hoàn chỉnh cho AI Agent của bạn.

### 1. 🐣 Bối cảnh: Đây là cái gì?

**Antigravity Awesome Skills** (Bản phát hành 5.0.0) là một bản nâng cấp khổng lồ cho khả năng của AI của bạn.

Các AI Agents (như Claude Code, Cursor, hoặc Gemini) rất thông minh, nhưng chúng thiếu các **công cụ cụ thể**. Chúng không biết "Giao thức triển khai" của công ty bạn hoặc cú pháp cụ thể cho "AWS CloudFormation".
**Skills** là các tệp markdown nhỏ dạy chúng cách thực hiện các tác vụ cụ thể này một cách hoàn hảo, mọi lúc.

### 2. ⚡️ Bắt đầu nhanh (1 phút)

Cài đặt một lần; sau đó sử dụng Starter Packs trong [docs/BUNDLES.md](docs/vietnamese/BUNDLES.vi.md) để tập trung vào vai trò của bạn.

1. **Cài đặt**:

   ```bash
   # Đường dẫn mặc định: ~/.agent/skills
   npx antigravity-awesome-skills
   ```

2. **Kiểm tra**:

   ```bash
   test -d ~/.agent/skills && echo "Skills đã được cài đặt tại ~/.agent/skills"
   ```

3. **Chạy skill đầu tiên của bạn**:

   > "Sử dụng **@brainstorming** để lập kế hoạch cho một SaaS MVP."

4. **Chọn một bundle**:
   - **Web Dev?** bắt đầu với `Web Wizard`.
   - **Security?** bắt đầu với `Security Engineer`.
   - **Sử dụng chung?** bắt đầu với `Essentials`.

### 3. 🧠 Cách sử dụng

Sau khi cài đặt, chỉ cần yêu cầu agent của bạn một cách tự nhiên:

> "Sử dụng skill **@brainstorming** để giúp tôi lập kế hoạch cho một SaaS."
> "Chạy **@lint-and-validate** trên tệp này."

👉 **[Đọc Hướng dẫn bắt đầu đầy đủ](docs/vietnamese/GETTING_STARTED.vi.md)**

Các skills này tuân theo định dạng **SKILL.md** phổ quát và hoạt động với bất kỳ AI coding assistant nào hỗ trợ agentic skills.

| Công cụ         | Loại | Ví dụ cách gọi                    | Đường dẫn         |
| :-------------- | :--- | :-------------------------------- | :---------------- |
| **Claude Code** | CLI  | `>> /skill-name help me...`       | `.claude/skills/` |
| **Gemini CLI**  | CLI  | `(User Prompt) Use skill-name...` | `.gemini/skills/` |
| **Codex CLI**   | CLI  | `(User Prompt) Use skill-name...` | `.codex/skills/`  |
| **Antigravity** | IDE  | `(Agent Mode) Use skill...`       | `.agent/skills/`  |
| **Cursor**      | IDE  | `@skill-name (trong Chat)`        | `.cursor/skills/` |
| **Copilot**     | Ext  | `(Dán nội dung thủ công)`         | N/A               |
| **OpenCode**    | CLI  | `opencode run @skill-name`        | `.agent/skills/`  |

> **Đường dẫn phổ quát**: Chúng tôi khuyên bạn nên clone vào `.agent/skills/`. Hầu hết các công cụ hiện đại (Antigravity, các CLI gần đây) đều tìm kiếm ở đây theo mặc định.

Để sử dụng các skills này với **Claude Code**, **Gemini CLI**, **Codex CLI**, **Cursor**, **Antigravity**, **OpenCode**, hoặc **AdaL**:

### Lựa chọn A: npx (khuyên dùng)

```bash
# Mặc định: ~/.agent/skills (phổ quát)
npx antigravity-awesome-skills

# Cursor
npx antigravity-awesome-skills --cursor

# Claude Code
npx antigravity-awesome-skills --claude

# Gemini CLI
npx antigravity-awesome-skills --gemini

# Codex CLI
npx antigravity-awesome-skills --codex

# OpenCode (Phổ quát)
npx antigravity-awesome-skills

# Đường dẫn tùy chỉnh
npx antigravity-awesome-skills --path ./my-skills
```

Chạy `npx antigravity-awesome-skills --help` để xem tất cả các tùy chọn. Nếu thư mục đã tồn tại, trình cài đặt sẽ chạy `git pull` để cập nhật.


## Xử lý sự cố

### `npx antigravity-awesome-skills` trả về lỗi 404

Sử dụng phương án dự phòng gói GitHub:

```bash
npx github:sickn33/antigravity-awesome-skills
```

### Vấn đề clone trên Windows (symlinks)

Repository này sử dụng symlinks cho các skills chính thức. Kích hoạt Developer Mode hoặc chạy Git với quyền Administrator, sau đó clone bằng:

```bash
git clone -c core.symlinks=true https://github.com/sickn33/antigravity-awesome-skills.git .agent/skills
```

### Skills đã cài đặt nhưng không được công cụ của bạn nhận diện

Cài đặt vào đường dẫn dành riêng cho công cụ (ví dụ `.claude/skills`, `.gemini/skills`, `.codex/skills`, `.cursor/skills`) hoặc sử dụng các cờ của trình cài đặt (`--claude`, `--gemini`, `--codex`, `--cursor`, `--path`).

### Cập nhật bản cài đặt hiện có

```bash
git -C ~/.agent/skills pull
```

### Cài đặt lại từ đầu

```bash
rm -rf ~/.agent/skills
npx antigravity-awesome-skills
```

## Bộ sưu tập tuyển chọn

**Bundles** là các nhóm skills được tuyển chọn cho một vai trò hoặc mục tiêu cụ thể (ví dụ: `Web Wizard`, `Security Engineer`, `OSS Maintainer`).

Chúng giúp bạn tránh việc phải chọn từng cái một từ hơn 700 skills.

Bundles là gì:

- Các bộ khởi đầu được đề xuất cho các workflows phổ biến.
- Một phím tắt để làm quen và thực thi nhanh hơn.

Bundles không phải là gì:

- Không phải là một bản cài đặt riêng biệt.
- Không phải là một bộ cài đặt sẵn bị khóa.

Cách sử dụng bundles:

1. Cài đặt repository một lần.
2. Chọn một bundle trong [docs/BUNDLES.md](docs/vietnamese/BUNDLES.vi.md).
3. Bắt đầu với 3-5 skills từ bundle đó trong prompt của bạn.
4. Chỉ thêm nhiều hơn khi cần thiết.

Ví dụ:

- Xây dựng một SaaS MVP: `Essentials` + `Full-Stack Developer` + `QA & Testing`.
- Gia cố production: `Security Developer` + `DevOps & Cloud` + `Observability & Monitoring`.
- Gửi các thay đổi OSS: `Essentials` + `OSS Maintainer`.

## Antigravity Workflows

Bundles giúp bạn chọn skills. Workflows giúp bạn thực hiện chúng theo thứ tự.

- Sử dụng bundles khi bạn cần các đề xuất được tuyển chọn theo vai trò.
- Sử dụng workflows khi bạn cần thực hiện từng bước cho một mục tiêu cụ thể.

Bắt đầu tại đây:

- [docs/WORKFLOWS.md](docs/WORKFLOWS_vi.md): các playbook dễ đọc cho con người.
- [data/workflows.json](data/workflows.json): metadata workflow có thể đọc được bằng máy.

Các workflows ban đầu bao gồm:

- Giao hàng một SaaS MVP
- Audit bảo mật cho một Web App
- Xây dựng một AI Agent System
- QA và Browser Automation (với tùy chọn hỗ trợ `@go-playwright` cho Go stacks)

## Cách đóng góp

Chúng tôi hoan nghênh những đóng góp từ cộng đồng! Để thêm một skill mới:

1. **Fork** repository.
2. **Tạo một thư mục mới** bên trong `skills/` cho skill của bạn.
3. **Thêm một `SKILL.md`** với frontmatter bắt buộc (tên và mô tả).
4. **Chạy xác thực**: `python3 scripts/validate_skills.py`.
5. **Gửi một Pull Request**.

Vui lòng đảm bảo skill của bạn tuân theo các thực hành tốt nhất của Antigravity/Claude Code.
