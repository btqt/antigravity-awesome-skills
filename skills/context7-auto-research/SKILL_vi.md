# context7-auto-research

## Tổng quan

Tự động lấy tài liệu thư viện/framework mới nhất cho Claude Code thông qua Context7 API

## Khi nào Sử dụng

- Khi bạn cần tài liệu cập nhật cho các thư viện và framework
- Khi hỏi về React, Next.js, Prisma, hoặc bất kỳ thư viện phổ biến nào khác

## Cài đặt

```bash
npx skills add -g BenedictKing/context7-auto-research
```

## Hướng dẫn Từng bước

1. Cài đặt kỹ năng bằng lệnh trên
2. Cấu hình khóa API (tùy chọn, xem kho lưu trữ GitHub để biết chi tiết)
3. Sử dụng tự nhiên trong các cuộc trò chuyện Claude Code

## Ví dụ

Xem [Kho lưu trữ GitHub](https://github.com/BenedictKing/context7-auto-research) để biết các ví dụ.

## Thực tiễn Tốt nhất

- Cấu hình khóa API thông qua biến môi trường để có giới hạn tốc độ cao hơn
- Sử dụng tính năng kích hoạt tự động của kỹ năng để tích hợp liền mạch

## Khắc phục sự cố

Xem kho lưu trữ GitHub để biết hướng dẫn khắc phục sự cố.
