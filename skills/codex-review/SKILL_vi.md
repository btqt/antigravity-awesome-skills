# codex-review

## Tổng quan

Đánh giá mã chuyên nghiệp với việc tạo CHANGELOG tự động, tích hợp với Codex AI

## Khi nào Sử dụng

- Khi bạn muốn đánh giá mã chuyên nghiệp trước khi cam kết (commits)
- Khi bạn cần tạo CHANGELOG tự động
- Khi đánh giá việc tái cấu trúc quy mô lớn

## Cài đặt

```bash
npx skills add -g BenedictKing/codex-review
```

## Hướng dẫn Từng bước

1. Cài đặt kỹ năng bằng lệnh trên
2. Đảm bảo Codex CLI đã được cài đặt
3. Sử dụng `/codex-review` hoặc trình kích hoạt ngôn ngữ tự nhiên

## Ví dụ

Xem [Kho lưu trữ GitHub](https://github.com/BenedictKing/codex-review) để biết các ví dụ.

## Thực tiễn Tốt nhất

- Giữ CHANGELOG.md trong thư mục gốc dự án của bạn
- Sử dụng thông báo cam kết quy ước (conventional commit messages)
